there is .gitignore and .gitkeep files 

1. ".gitignore" = it is used to ignore the file which we dont want to track it and then also git showing the message to add 

2. ".gitkeep" = it is file used for empty folder which git dont tracks and a possibility of getting the folder lost for that this is used you can add this file for this will make git show message of adding the folder and you can track that folder
